Movie Seat Booking - En webbapp där användaren kan välja biosalongens säten och se totalpriset baserat på vald film. 
Filmerna hämtas från ett REST API via JSON Server.


- Välj film från en dropdown — filmer hämtas dynamiskt från API:t
- Klicka på lediga sittplatser för att välja/avvälja dem
- Se antal valda platser och totalpriset uppdateras automatiskt
- Val sparas i localStorage och finns kvar vid omladdning


Node.js installerat, 
npm installerat


HTML / CSS / JavaScript

JSON Server
